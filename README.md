# Market Signal Intelligence — Trading Analysis App

A full-stack trading analysis platform with AI-driven signals, paper trading, risk calculation, and a trade journal. Built with a Node.js/Express backend and a React (Vite + Tailwind) frontend.

---

## Project Structure

```
CLAUDE CODE/
├── backend/      # Node.js + Express REST API (SQLite via better-sqlite3)
└── frontend/     # React 18 + Vite + Tailwind CSS SPA
```

---

## Prerequisites

- **Node.js** v18 or later
- **npm** v9 or later

---

## Backend Setup

```bash
cd backend
npm install
```

### Configure environment variables

Copy the example file and fill in your values:

```bash
cp .env.example .env
```

Edit `.env`:

```
PORT=3001
JWT_SECRET=change_this_to_a_long_random_secret
NODE_ENV=development
TWELVE_DATA_API_KEY=your_key_here        # optional — for live market data
ALPHA_VANTAGE_API_KEY=your_key_here      # optional — for live market data
BINANCE_API_KEY=your_key_here            # optional — for crypto data
```

> The app ships with mock market data, so API keys are not required to run locally.

### Run the backend

Development (auto-reload):
```bash
npm run dev
```

Production:
```bash
npm start
```

The API will be available at `http://localhost:3001`.

---

## Frontend Setup

```bash
cd frontend
npm install
```

### Run the frontend

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

### Build for production

```bash
npm run build
```

Serve the `dist/` folder with any static file host.

---

## Running Both Together

Open two terminals:

**Terminal 1 — Backend:**
```bash
cd backend && npm run dev
```

**Terminal 2 — Frontend:**
```bash
cd frontend && npm run dev
```

Then open `http://localhost:5173` in your browser.

## Features

- **Dashboard** — market overview, top movers, and recent signals
- **Signal Centre** — buy/sell/hold signals generated by technical indicators (RSI, MACD, Bollinger Bands, etc.)
- **Asset Detail** — charts and deep-dive analysis per asset
- **Watchlist** — track your favourite symbols
- **Screener** — filter assets by signal strength and other criteria
- **Paper Trading** — simulate trades without real money
- **Trade Journal** — log and review past trades
- **Risk Calculator** — position sizing and risk/reward analysis
- **Strategy Tester** — backtest simple strategies against historical data
- **Alerts** — price and signal-based notifications
- **Education** — guides on technical analysis concepts
- **Settings** — user preferences and account management
